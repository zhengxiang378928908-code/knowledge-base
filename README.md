# Knowledge Base

个人知识库，使用 Obsidian 管理。

## 仓库定位

这个仓库主要用于沉淀以下内容：

- Java 面试备战资料
- 工具使用指南
- 可持续维护的个人知识卡片

内容以 Markdown 为主，优先保证可读性、可检索性和后续可扩展性。

## 使用方式

用 Obsidian 打开本目录即可。

## 维护约定

- 优先复用已有文档，避免创建语义重复的笔记
- 成体系的内容尽量沿用现有编号方式，例如 `00-目录`、`01-主题`、`02-主题`
- 速查类内容放到对应主题下的 `知识点速查/`
- 原始资料、简历、导出内容放到 `面试资料/`
- 仅提交需要共享的 `.obsidian` 配置，不提交本地工作区状态

## 协作说明

仓库根目录下的 `agent.md` 记录了面向 AI / Agent 协作时的上下文、编辑边界和整理规范。后续如果继续让助手维护这个知识库，可以直接把它作为默认协作说明使用。
