## 复习建议

1. **第一轮**：精读 01（项目深挖），把每个问题的答案用自己的话说一遍
2. **第二轮**：刷 02-06（八股文），重点理解原理，结合项目场景记忆
3. **第三轮**：看 07-08（AI + 系统设计），根据目标岗位侧重
4. **第四轮**：过一遍 09-10（设计模式 + 软技能），准备 STAR 故事
5. **考前**：重点回顾 01 + 11 + 12，确保两个核心项目都能讲到代码级细节
